HelpGUI - Icons Themes

Help GUI includes 9 icons themes as:



BlueCurve, from the Linux BlueCurve theme.


Crystal, from the KDE 3 theme.


Gtk, from Gtk stock theme. (use gtk or gtk2 for 16x16 icons)


Java, from swing java look'n'feel.


Jfa, from jfa icon project by by Dean S. Jones (dean@gallant.com).


Kde, from Kde project.


Plastic, from JGoodies plastic Theme by Karsten Lentzsch (info@JGoodies.com).


Slick, from Kde Project.


Windowsfrom Microsoft Project.
